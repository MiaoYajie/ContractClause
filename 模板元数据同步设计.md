## 源-法天使API服务

### 模板列表API

```curl
curl -X 'GET' \
  'https://contentadmin.fatianshi.cn/content/ForReview/FindUpdatedTemplates?size=50&page=0&recommend=true' \
  -H 'accept: */*' \
  -H 'Authorization: Bearer 69a0004e1643b5570163291026b0309a'
```

参数说明：
page: 页码从0开始
size: 每页读取条数
recommend：固定true，只同步推荐模板
Authorization Token: 从appsetting中读取

## 同步策略
每天同步一次

**读取数据逻辑**
已知api 获取的数据列表排序是更新时间倒叙排列
因此 首先读取数据库表templates中最大的更新时间（SourceUpdatedAt）的值
循环调用API 迭代页码，每页读取50条，直到api返回的数据中存在PublishedOn小于等于最大更新时间的数据为止
如果没有SourceUpdatedAt，则将api获取到的所有数据都同步


## 数据处理

api返回的数据 跟 Template实体 字段映射

### 模板元数据处理（API Item -> Template 实体映射）

Id -> Id
Number -> Number
Title -> Title
Title2 -> Alias
TemplateTypeName -> Type
TemplateCategoryNames -> Categories
ContentTag + VersionTag + PartyTag -> Tags
Brief -> Summary
SceneFor（用空格拼接成字符串） -> Scenarios
PublishedOn -> SourceUpdatedAt
true -> IsOfficial
null -> OwnerId
Version -> Version
当前时间 -> CreatedAt
false -> IsDeleted

**说明**
Alias 别名字段需要新增
ContentHtml、ContentMarkdown、ExternalId 字段不再使用需要删除
